# Gnaanesh Kalidevashankar — Portfolio

Premium dark-themed portfolio built with React + Vite + Tailwind CSS + Framer Motion.

## Stack
- React + Vite
- Tailwind CSS (v4 via @tailwindcss/vite)
- Framer Motion
- React Icons
- Web3Forms (contact form, no backend)

## Setup
npm install
npm run dev

## Deploy
npm run build
# drag dist/ folder to netlify.com

## Customize
All content is centralized in: src/constants/data.js
- Edit HERO, ABOUT, SKILLS, PROJECTS there
- Replace public/photo.jpg with your photo
- Replace WEB3FORMS_KEY with your key from web3forms.com
